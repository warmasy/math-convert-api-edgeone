💡 使用说明：
1. 本地测试：保持默认 http://localhost:3000，先运行 npm run dev
2. EdgeOne 部署后：将上方地址改为你的 EdgeOne 域名，如 https://api.example.com
3. GET 接口可直接点击「新标签页打开」测试；POST 接口使用下方按钮测试
